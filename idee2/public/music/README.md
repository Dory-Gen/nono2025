# Ajouter de la Musique de Noël

Pour ajouter de vraies chansons de Noël au lecteur audio :

## Option 1 : Télécharger depuis Incompetech (Recommandé)

1. Visitez https://incompetech.com/music/royalty-free/music.html
2. Recherchez "Christmas" dans la barre de recherche
3. Téléchargez ces chansons (format MP3) :
   - "Jingle Bells"
   - "We Wish You a Merry Christmas"
   - "Deck the Halls"
   - "Silent Night"
   - "Carol of the Bells"

4. Renommez les fichiers :
   - `jingle-bells.mp3`
   - `we-wish-you.mp3`
   - `deck-the-halls.mp3`
   - `silent-night.mp3`
   - `carol-of-bells.mp3`

5. Placez-les dans le dossier `public/music/`

## Option 2 : Utiliser des URLs publiques

Vous pouvez aussi utiliser des URLs directes vers des fichiers MP3 hébergés en ligne (assurez-vous qu'ils sont libres de droits).

## Note

Les fichiers actuels dans le lecteur sont des fichiers de test génériques. 
Une fois que vous aurez ajouté de vrais fichiers MP3 de Noël dans `public/music/`, 
le lecteur les utilisera automatiquement.
